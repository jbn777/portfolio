Assignment 1: GitHub Pages & Website Creation

Objective

The objective of this assignment was to create a portfolio website using GitHub Pages, understand the basics of Git and GitHub, and document the learning process.

Creating GitHub Pages

I created a GitHub repository named angelsnelson.github.io and enabled GitHub Pages from the repository settings. This allowed the repository to be hosted as a static website.

Website Creation

Multiple HTML files were created to structure the website, including a home page, an about page, an assignment documentation page, and a final project page. A navigation menu was added to connect all pages.

Learning Git Basics

I learned basic Git operations such as creating repositories, adding files, committing changes, and pushing updates to GitHub. These commands help in version control and project tracking.

What I Learned

  • How to host a website using GitHub Pages
  • Basic HTML page structure
  • Using GitHub for version control
  • Importance of documentation

Conclusion

This assignment helped me understand how GitHub can be used both as a version control system and as a platform to host personal documentation websites.
